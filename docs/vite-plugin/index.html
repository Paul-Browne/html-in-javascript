HT.JS

Vite plugin

HT.JS has its very own full-featured Vite plugin

vite-plugin-html-pages is a Vite plugin that allows you to easily create HTML pages from JavaScript files.

npm install vite vite-plugin-html-pages javascript-to-html

Installation

import { defineConfig } from 'vite';
import htmlPages from 'vite-plugin-html-pages';

export default defineConfig({
    base: '',
    plugins: [
        htmlPages({
            pagesDir: 'src',
            cleanUrls: true,
            debug: true
        }),
    ],
    build: {
    emptyOutDir: true,
    outDir: 'build',
    },
    server: {
    port: 3000,
    host: 'localhost',
    },
});
